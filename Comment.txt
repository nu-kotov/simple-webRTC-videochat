Реализовать соединение peer to peer и передачу видеопотока с помощью webRTC.

1) Установить node.js с https://nodejs.org/en/
2) В в папке с проектом в консоли создать файл package.json командой npm init (в проекте уже приложен)
3) Установить модули budo и watchify командой npm install budo warchify
4) Запустить сервер командой npm start
5) Инициатор соединения http://localhost:9966/#init, на его стороне формируется оффер, который необходимо копировать в поле other ID принимающей стороны, где сгенерируется оффер, который необходимо копировать в поле other ID инициатора
6) Браузер спросит согласие на включение камеры, согласиться

В данном решении метод getUserMedia инициализируется для Chrome, работает на локальном сервере
